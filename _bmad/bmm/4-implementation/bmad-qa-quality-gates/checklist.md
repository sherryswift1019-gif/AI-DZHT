# QG Quality Gates — Validation Checklist

## Data Collection

- [ ] Test results loaded (or baseline mode noted)
- [ ] Coverage data loaded (or noted as unavailable)
- [ ] Flake history considered
- [ ] Previous gate results compared (if available)

## Assessment

- [ ] All 5 dimensions evaluated
- [ ] Thresholds applied correctly (hard vs soft)
- [ ] Go/No-Go decision matches actual metrics
- [ ] No hard threshold failure marked as "pass"

## Action Items

- [ ] Every failing dimension has specific fix items
- [ ] Effort estimates provided
- [ ] Priority order makes sense (fix blockers first)

## Output

- [ ] Report saved to configured output path
- [ ] Decision clearly stated at top of report
- [ ] Metrics table complete with all dimensions

---

**Quality Bar**: The report should allow a PM to make a ship/no-ship decision in under 2 minutes.
