## 3. 数据流

```
SQLite requirements_table
  │  status ∈ {queued, running, blocked, done}
  │  pipeline: JSON[{status: PipelineStepStatus, ...}]
  ▼
GET /api/v1/projects/{id}/requirements        ← FastAPI 后端
  ▼
useRequirementList(projectId)                  ← TanStack Query (5s refetch when running)
  ▼
useMemo → statCounts                           ← 纯前端 O(n) 遍历
  │  { queued, running, pendingApproval, pendingInput, blocked, done }
  │  恒等式: sum(6桶) === requirements.length
  ▼
统计栏 JSX                                      ← 本次变更范围
  ├── 介入组: StatChip × 3 (待审批/阻塞/待输入)
  ├── 进度组: StatChip × 3 (执行中/待执行/已完成)
  └── 总数标签
```

## 4. 当前统计分类逻辑（不变）

```typescript
// ProjectDetailPage.tsx:130-151
const statCounts = useMemo(() => {
  const counts = { queued: 0, running: 0, pendingApproval: 0, pendingInput: 0, blocked: 0, done: 0 }
  for (const r of requirements) {
    switch (r.status) {
      case 'done':    counts.done++;    break
      case 'blocked': counts.blocked++; break
      case 'queued':  counts.queued++;  break
      case 'running': {
        const current = r.pipeline.find(s => s.status !== 'done')
        if (current?.status === 'pending_approval' || current?.status === 'pending_advisory_approval') {
          counts.pendingApproval++
        } else if (current?.status === 'pending_input') {
          counts.pendingInput++
        } else {
          counts.running++
        }
        break
      }
    }
  }
  return counts
}, [requirements])
```

**分类恒等式**：`pendingApproval + blocked + pendingInput + running + queued + done === requirements.length`

## 6. 当前 StatChip 组件（变更目标）

```tsx
// ProjectDetailPage.tsx:1320-1327
function StatChip({ label, value, color }: { label: string; value: number; color: string }) {
  return (
    <div className="flex flex-col items-center gap-0.5 min-w-[48px]">
      <span className="text-xl font-bold tabular-nums" style={{ color }}>{value}</span>
      <span className="text-[10px] text-[var(--text-3)]">{label}</span>
    </div>
  )
}
```

**当前样式特征**：数字 `text-xl`（20px）加粗 + 标签 `10px` 灰色，纵向排列，最小宽度 48px。
